Git

  • git log - Команда "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • git checkout - Команда "сменить папку" или "перейти в другую папку".
  • git status - Команда показывает, в какой сейчас папке мы находимся.
  • git dir - расшифровывается как "Make directory", что переводится как "создать папку".
  • git mv - расшифровывается как"Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать или удалить файл/
